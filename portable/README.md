# Портативная версия PDM System 0.22.0

Готовый `.exe` весит больше 100 МБ, а GitHub не принимает такие файлы,
поэтому он разбит на части.

## Как собрать и запустить (Windows)
1. Скачайте все файлы `PDM-System-0.22.0.exe.XX.part` из этой папки
   в одну папку на компьютере.
2. Запустите **`СОБРАТЬ-EXE.bat`** (двойной клик) — он склеит части в
   `PDM System 0.22.0.exe`.
3. Запустите получившийся `PDM System 0.22.0.exe`.

Установка не требуется — это портативная версия.

## Что нового в 0.22.0
- **Раздел «Настройки»** в левой панели (над профилем): категории как в
  настройках Windows/iOS — Общие, Менеджмент, Оборудование, Уведомления,
  База данных, Crash-логи, Обновления. Настройки из профиля перенесены сюда.
- **Менеджмент**: свои этапы закупки (название, значок, цвет, порядок —
  «Настройки → Менеджмент»); массовая смена этапа нескольким позициям
  (галочки или Ctrl+клик); режим «Дерево» — родительский тег и его дочерние;
  фильтр по отделу и сортировка по количеству.
- **Теги**: исправлен лаг поиска и «зависшее» выделение после удаления
  тега; «Поделиться в чате» и «Связи» по ПКМ теперь работают и в
  «Дереве связей», и в «Спецификации» (Ctrl+клик — несколько).
- **Блокнот**: шрифты, ссылки, чек-листы с галочками, горизонтальная линия,
  вставка даты, поиск по заметке, экспорт в Word (.doc) и печать.
- **Журнал действий**: записывается каждый клик, фокус и ввод в поля,
  ошибки JS — по журналу сразу видно, из-за чего возникла проблема.
- **Быстродействие**: исправлено накопление журнала в памяти (программа
  фризила со временем), быстрая проверка базы при запуске (программа
  открывается заметно быстрее на больших базах).
- **Срок действия профиля** теперь проверяется по надёжному времени:
  якорь «максимального замеченного времени» + сетевое время. Перевод
  часов назад блокирует вход просроченных профилей, а не продлевает их.
