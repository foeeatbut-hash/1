# Портативная версия PDM System 0.20.0

Готовый `.exe` весит ~109 МБ, а GitHub не принимает файлы больше 100 МБ,
поэтому он разбит на две части.

## Как собрать и запустить (Windows)
1. Скачайте оба файла из этой папки в одну папку на компьютере:
   - `PDM-System-0.20.0.exe.00.part`
   - `PDM-System-0.20.0.exe.01.part`
2. Запустите **`СОБРАТЬ-EXE.bat`** (двойной клик) — он склеит части в
   `PDM System 0.20.0.exe`.
   - Либо вручную в командной строке:
     `copy /b PDM-System-0.20.0.exe.00.part+PDM-System-0.20.0.exe.01.part "PDM System 0.20.0.exe"`
3. Запустите получившийся `PDM System 0.20.0.exe`.

Установка не требуется — это портативная версия.

## Что нового в 0.20.0
- Окно программы: без надписи сверху, скруглённые углы, аккуратные кнопки
  свернуть / развернуть / закрыть.
- Рабочий чат перекрашен под единый изумрудный стиль программы.
- Полностью переделан импорт тегов из таблиц: мастер с выбором файла из
  Проводника, переключением вкладок Excel снизу, разметкой колонок
  (код / марка / наименование / отдел / среда / WBS / родитель / актуальность)
  с авто-определением, объединением или дублированием при совпадении кодов.
- Дерево связей: кнопка «Упорядочить» — авто-раскладка без наложений,
  дубликаты тегов подсвечиваются.
