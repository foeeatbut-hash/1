# Портативная версия PDM System 0.21.0

Готовый `.exe` весит ~109 МБ, а GitHub не принимает файлы больше 100 МБ,
поэтому он разбит на две части.

## Как собрать и запустить (Windows)
1. Скачайте оба файла из этой папки в одну папку на компьютере:
   - `PDM-System-0.21.0.exe.00.part`
   - `PDM-System-0.21.0.exe.01.part`
2. Запустите **`СОБРАТЬ-EXE.bat`** (двойной клик) — он склеит части в
   `PDM System 0.21.0.exe`.
   - Либо вручную в командной строке:
     `copy /b PDM-System-0.21.0.exe.00.part+PDM-System-0.21.0.exe.01.part "PDM System 0.21.0.exe"`
3. Запустите получившийся `PDM System 0.21.0.exe`.

Установка не требуется — это портативная версия.

## Что нового в 0.21.0 (граф «Дерево связей» / Dynamo)
- **Поиск**: универсальное поле поиска по разделу — код тега, наименование,
  марка, отдел, среда, WBS; спец-запрос «дубли» показывает только дубликаты.
  Результаты — в выпадающем списке с галочками для мультивыбора; клик —
  перейти и выделить, «Показать выбранные» — выделить сразу несколько.
- **Карточки холста**: не перекрывают соседей при разворачивании (разворот
  происходит поверх, z-index, без сдвига раскладки). Всегда видно тег,
  наименование, марку и актуальность; дубликаты — красная рамка и бейдж.
- **Производительность**: отсечение по видимой области (viewport culling) и
  отключение анимации связей при большом количестве тегов — граф с сотнями
  тегов не тормозит.
- **Упорядочение**: полностью переделано — независимые деревья упаковываются
  в компактную сетку вместо одной длинной «простыни», переключатель
  ориентации ⇅/⇄, автоматическое вписывание в экран после раскладки.
  Если что-то выделено — упорядочивается и вписывается только эта ветвь.
- **Управление холстом**: кнопка «Центрировать» — один клик центрирует
  выделение (или даёт выбрать «главного родителя», если корней несколько),
  двойной клик — полная авто-раскладка всего холста.
- **Связи по ПКМ**: правый клик по карточке — меню «Связи»: выделить
  родителей (вверх по цепочке), потомков (вниз), всю ветвь целиком, а также
  «Упорядочить ветвь» и «Поделиться» — в одном меню.
