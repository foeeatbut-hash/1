# Портативная версия PDM System 0.21.0

Готовый `.exe` весит больше 100 МБ, а GitHub не принимает такие файлы,
поэтому он разбит на части.

## Как собрать и запустить (Windows)
1. Скачайте все файлы `PDM-System-0.21.0.exe.XX.part` из этой папки
   в одну папку на компьютере.
2. Запустите **`СОБРАТЬ-EXE.bat`** (двойной клик) — он склеит части в
   `PDM System 0.21.0.exe`.
   - Либо вручную в командной строке:
     `copy /b PDM-System-0.21.0.exe.00.part+PDM-System-0.21.0.exe.01.part "PDM System 0.21.0.exe"`
3. Запустите получившийся `PDM System 0.21.0.exe`.

Установка не требуется — это портативная версия.

## Что нового в 0.21.0
- **Новый раздел «Менеджмент»** — закупки поверх той же базы тегов:
  жизненный цикл позиции Добавлен → Заказан → Утверждён → Куплен с датами
  и автором каждого этапа, поставщик/кол-во/примечание, счётчики-фильтры,
  поиск и фильтры «только дубли» / «требуют внимания».
- **Динамо-холст (Теги)**: универсальный поиск с выпадающим списком
  (тег/название/марка/дубли, клик — центрирование, галочки — мультивыбор);
  в карточке всегда видны марка и актуальность; новые карточки не
  перекрывают существующие; рендер только видимой области на больших
  холстах (не лагает); «Центрировать» — 1 клик выбирает главного родителя
  и центрирует его дерево, 2 клика вписывают весь холст; ПКМ по карточке —
  «Связи» вверх/вниз по цепочке и «Поделиться в рабочем чате»
  (Ctrl+клик — несколько тегов одним сообщением, каждая кнопка — название
  тега, у получателя клик открывает граф и подсвечивает карточку).
- **Проводник**: два встроенных раздела «Общий» (виден всем) и «Личный»
  (только владельцу); разделы нельзя удалить или переименовать; Главный
  Администратор видит личные папки всех пользователей.
