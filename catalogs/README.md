# Каталоги и бланки оборудования ВЕЗА

Сюда складываются исходные материалы, из которых собирается **база знаний**
оборудования ВЕЗА (категории, марки, типоразмеры, характеристики, правила
комплектации «что с чем» и правила по отделам).

## Куда что класть

| Папка | Что класть | Зачем |
|-------|-----------|-------|
| `catalogs/` | **Каталоги и типоразмерные ряды** — где перечислены *все* модели и диапазоны параметров (а не один экземпляр) | Наполнение базы: категория → семейство → типоразмер → характеристики |
| `catalogs/blanks/` | **Бланки-заказы / опросные листы** — один уже подобранный аппарат под проект | Правила «что с чем идёт» (вентилятор + соединитель + виброопоры …) и правила по отделам |

## Как загрузить (проще всего)

На сайте GitHub в этой папке: **Add file → Upload files** → перетащить файлы → **Commit changes**.
Веб-загрузка тянет примерно до 25 МБ на файл. Файл крупнее — бей на части или
заливай в Excel (он легче).

## В каком виде лучше присылать

1. **Excel (.xlsx) / CSV** — идеально: точные числа и единицы, чистый разбор.
2. **.docx** — хорошо (бланки-заказы как раз такие).
3. **PDF** — читается, в т.ч. «шрифтовые» бланки ВЕЗА (через ToUnicode).
4. **Скриншоты страниц (PNG/JPG)** — тоже читаются (как картинки), удобно для пары таблиц.

Имя файла — лучше короткое латиницей (`osa_evo.xlsx`), но кириллица тоже допустима.

## Что дальше

После загрузки нового файла напиши в чат «залил каталог …» —
я заберу его (`git pull`), разберу и наполню базу, а структуру таблицы
покажу на подтверждение.
